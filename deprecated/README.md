# Dayton Johnson

https://www.daytonrjohnson.com

## Notable Technologies

- Next.JS / React.JS
- TypeScript

## Scripts

- `yarn dev`: Starts the development server
- `yarn start`: Starts the production server
- `yarn build`: Builds a production static version of the website
